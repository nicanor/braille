# Braille

Braille es una gema creada para realizar pruebas de traducción Braille básicas, para aprender un poco más sobre lectura Braille.

## Instalación

Add this line to your application's Gemfile:

    gem 'braille'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install braille
